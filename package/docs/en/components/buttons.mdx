# Buttons

The **Btn** component replaces the standard html button with a ManaKit theme and a multitude of options. Any color helper class can be used to alter the background or text color.

<div className="d-flex justify-space-between">
  <Btn>Button</Btn>
  <Btn stacked prependIcon={<Icon>{mdiHome}</Icon>}>Button</Btn>
  <Btn stacked appendIcon={<Icon>{mdiHome}</Icon>}>Button</Btn>
  <Btn prependIcon={<Icon>{mdiHome}</Icon>}>Button</Btn>
  <Btn appendIcon={<Icon>{mdiHome}</Icon>}>Button</Btn>
</div>

### Looader

Using the loading prop, you can notify a user that there is processing taking place. The default behavior is to use a **ProgressCircular** component but this can be customized with the loader slot.

<div className="d-flex justify-space-between">
  <Btn loading={true}>Default</Btn>
</div>

Buttons can be given different sizing options to fit a multitude of scenarios.
